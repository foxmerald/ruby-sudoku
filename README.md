# Project Title

Ruby Sudoku

## Description

A Sudoku Terminal Game built in pure Ruby.

## Getting Started

### Prerequisites 

* [ruby](https://www.ruby-lang.org/en/documentation/installation/)

### Installation

* Clone the repo
  ```sh
  git clone 
  ```

### Executing program

* Run with 
  ```sh
  bundle exec ruby main.rb
  ```
  or simply
  ```sh
  ./play
  ```

## How to Play
Commands within the game:

| Key        | Command                                 |
| ---------- | --------------------------------------- |
| arrow keys | Move cursor                             |
| 1-9        | Enter a number in an empty field        |
| c          | Submit your current solution            |
| x          | Delete your input in the selected field |
| n          | Start a new game                        |
| q          | Quit the game                           |
|            |                                         |

## Game Modes

Default mode is moderate. You can choose the difficulty at the start of the game.

| Key | Level    |                     |
| --- | -------- | ------------------- |
| e   | Easy     | Good for beginners  |
| m   | Moderate | Balanced experience |
| h   | Hard     | More challenging    |
